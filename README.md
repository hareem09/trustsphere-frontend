# trustsphere-frontend
TrustSphere - Trust Organization Management System
TrustSphere is a frontend-based Trust Organization Management System built using React JS. The application helps trust organizations manage donations, beneficiaries, events, volunteers, and organizational settings through a professional dashboard interface.

Dashboard Overview
The dashboard provides an overview of total donations, active beneficiaries, upcoming events, and quick statistics using cards and charts.
CRUD Operations
The system supports Create, Read, Update, and Delete operations for donations, beneficiaries, and events.
Routing
React Router DOM is used for multi-page navigation between all sections of the application.
Data Persistence
All data is stored in browser local storage to simulate backend persistence.
